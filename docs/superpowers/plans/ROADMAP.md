# Excel Diff 功能完善路线图

## 概述
基于 Beyond Compare 功能对比分析，按照优先级完善我们的 Excel 对比工具。

---

## P0 - 核心完善（高优先级）

### Task 1: 添加文件修改日期显示
**文件：**
- `src-tauri/src/commands/mod.rs` - 读取文件时包含修改时间
- `src/types/excel.ts` - 扩展类型定义
- `src/components/FileList.tsx` - 显示修改时间

**实现细节：**
1. 在 `FileEntry` 类型中添加 `modifiedAt?: number` 字段
2. 修改 `list_excel_files` 命令读取文件修改时间
3. 在文件列表中显示修改时间（格式：`2024-05-31 14:30`）
4. 对齐显示在文件大小旁边

---

### Task 2: 添加行号显示
**文件：**
- `src/components/DiffView.tsx` - 添加行号列
- `src/components/DiffGrid.tsx` - 渲染行号

**实现细节：**
1. 在左右两侧表格最左边添加行号列
2. 行号列固定宽度，灰色背景
3. 显示原始 Excel 中的行号（不是视图索引）
4. 行号与数据行同步滚动

---

### Task 3: 添加差异图例
**文件：**
- `src/components/DiffView.tsx` - 添加图例组件

**实现细节：**
1. 在对比视图右上角添加可收起的图例面板
2. 图例项：
   - 🟢 绿色背景 - 新增行（仅右侧）
   - ⚪ 灰色背景 - 删除行（仅左侧）
   - 🔴 浅红背景 - 修改行
   - 🔴 深红单元格 - 修改的单元格
3. 点击图例标题可以收起/展开

---

## P1 - 体验优化（中优先级）

### Task 4: 文件名过滤功能
**文件：**
- `src/components/FileList.tsx` - 添加过滤输入框

**实现细节：**
1. 在工具栏添加过滤输入框
2. 支持通配符（`*.xlsx`、`*report*` 等）
3. 实时过滤文件列表
4. 保留当前的状态过滤（全部/不同/相同/仅左/仅右）
5. 两种过滤条件同时生效

---

### Task 5: 查找功能
**文件：**
- `src/components/DiffView.tsx` - 添加查找工具栏
- `src/components/DiffGrid.tsx` - 支持高亮匹配项

**实现细节：**
1. 添加 `Ctrl+F` 快捷键打开查找栏
2. 查找栏包含：输入框、上一个/下一个按钮、关闭按钮
3. 高亮显示所有匹配项
4. 当前匹配项用不同颜色高亮
5. 支持在左右两侧同时查找

---

### Task 6: 历史记录命名
**文件：**
- `src/stores/historyStore.ts` - 添加 name 字段
- `src/components/HistoryBar.tsx` - 支持重命名

**实现细节：**
1. 在 `HistoryRecord` 中添加可选的 `name` 字段
2. 点击历史记录项时可以编辑名称
3. 如果没有命名，显示目录路径缩略
4. 双击历史记录名称进入编辑模式

---

## P2 - 高级功能（低优先级）

### Task 7: 文件夹树状视图
**文件：**
- `src/components/FileList.tsx` - 重构为树状组件

**实现细节：**
1. 替换当前的扁平分组为真正的树状结构
2. 支持多级嵌套文件夹
3. 文件夹图标与文件图标区分
4. 保持当前的同步展开功能

---

### Task 8: 单元格格式差异
**文件：**
- `src-tauri/src/excel/reader.rs` - 读取格式信息
- `src/types/excel.ts` - 添加格式类型
- `src/utils/diffEngine.ts` - 比较格式差异
- `src/components/DiffGrid.tsx` - 显示格式差异提示

**实现细节：**
1. 读取单元格的字体、颜色、边框、填充格式
2. 在差异比较时同时比较格式
3. 格式差异用另一种颜色标记（如橙色边框）
4. 鼠标悬停显示具体格式差异详情

---

### Task 9: 会话管理
**文件：**
- `src/stores/sessionStore.ts` - 新建会话状态管理
- `src/components/SessionManager.tsx` - 会话管理界面

**实现细节：**
1. 保存完整的会话状态（包括展开的文件夹、选择的关键列等）
2. 会话列表显示在侧边栏或菜单中
3. 支持将会话添加到收藏夹
4. 支持会话导入/导出为 JSON 文件

---
